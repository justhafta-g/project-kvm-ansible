LoadBalancer role
=========

Povisioning NGINX on KVM host for balance trafic between VMs

Role Variables
--------------

Before use role needs to define variables in the vault:

 -     MYSQL_ROOT_PASSWORD
 -     MYSQL_DATABASE
 -     MYSQL_USER
 -     MYSQL_PASSWORD
 -     WORDPRESS_DB_HOST
 -     WORDPRESS_DB_USER
 -     WORDPRESS_DB_PASSWORD
